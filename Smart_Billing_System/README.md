# Smart Billing System


A Python-based billing system that supports item entry, discount rules, tax calculation, and PDF invoice generation.

### Project Structure

main.py – main controller of the billing program



modules/
  billing.py – billing calculations
  discount.py – discount logic
  tax.py – tax calculation
  input_handler.py – user input handling
  pdf_generator:-bill generating logic
  

data/
  discount_rules.json – discount configuration
  tax_rules.json - tax configurations

output/
INV-0001.pdf - output
